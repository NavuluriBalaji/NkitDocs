class nkit.agent.Agent

The ReAct Engine

NKit agents are autonomous solvers. Built on the ReAct (Reasoning and Acting) loop, they recursively execute THINK -> ACT -> OBSERVE logic chains. They automatically parse outputs, invoke Python sandbox tooling, and evaluate their own hallucinations against system invariants.

# Standard autonomous agent injection
from NKit import Agent

agent = Agent(
    name="SystemArchitect",
    model="gpt-4-turbo",
    temperature=0.2, # Analytical rigidity
    max_iterations=15,
    enable_memory=True
)

await agent.aprocess(task="Re-write the microservice map.")
class nkit.crews.Crew

Multi-Agent Swarms

Scale single instances into massive autonomous pipelines. NKit Crews orchestrate multiple specialized agents using execution strategies: Sequential Pipelines, Hierarchical Delegation, or Parallel Execution. All context is shared dynamically via the internal Memory Registry.

Sequential

A -> B -> C execution.

Hierarchical

module nkit.memory.*

State Management

Stateless execution is useless for production. Agents hook into ConversationMemory, VectorMemory, or generic FileMemory to cache entities. NKit prevents prompt bloat by automatically tracking token footprints and intelligently pruning deep history.

  • Conversation state keeps JSON tracks of every LLM interaction.
  • Entity memory tracks isolated variables and metadata payloads.
  • Token Pruning forces cache invalidation using agent.memory.prune() once token limit hits peak density.
class nkit.tools.Tool

Capability Hooks

Native zero-barrier extensions. NKit converts raw Python functions securely into strict JSON schemas formatted appropriately for any registered LLM backend. If the schema validation fails, the native CodeACT loop corrects the malformed payload without crashing securely.

from NKit.tools import Tool

def sys_execute(cmd: str) -> str:
    """Runs verified OS commands on node."""
    return subprocess.check_output(cmd)

sandbox_tool = Tool(
    name="sys_execute",
    callback=sys_execute
)
agent.add_tool(sandbox_tool)
